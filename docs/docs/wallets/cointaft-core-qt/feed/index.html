]]> How to Install CoinTAFT Wallet ?

Download CoinTAFT QT Wallet

 

Windows Installer (.exe)

Download the latest CoinTAFT CORE release for Windows, it will end with “cointaft-win-x.x.x.x.zip“ and extract the zip file and you get file with name “cointaft-qt.exe”

Execute the .exe by double clicking or right click > open.

Wait for cointaft core to fully synchronize. You can hover over TAFT icon at bottom right to see if it is synchronized. Not complete looks like this:

Get Address 

After cointaft core to fully synchronize, click Receive and click Request Payment

 

Optionals

You can greatly increase synchronization speed by using  the bootstraps the CoinTAFT provides. If this is your first time installing the wallet you first need to create the CointaftCore directory.

Unix/Linux/Mac: $HOME/.CointaftCore
Windows: %AppData%/CointaftCore

 

CointaftCore Security

By default CointaftCore does not encrypt the wallet.dat files, it is very important that you enable encryption on your wallet. Otherwise anybody that has your wallet.dat or access to your wallet can take your TAFT. You should read CointaftCore Security at your earliest convenience.
